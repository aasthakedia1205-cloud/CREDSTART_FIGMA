#  CREDSTART - UI/UX Design (Figma AI)

##  Overview
This project showcases a modern UI/UX design prototype created using Figma AI tools, along with its HTML implementation.
The design is inspired by fintech applications like CRED, focusing on a clean, minimal, and user-friendly interface.

##  Features
- Modern fintech-inspired UI  
- Clean and minimal design  
- User-friendly navigation  
- Consistent color scheme and layout
- Responsive HTML layout

##  Tools Used
- Figma  
- Figma AI
- HTML5
- CSS3
